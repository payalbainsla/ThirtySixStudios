ThirtySixStudio Clone 🎨✨

A modern, responsive, and animated clone of the ThirtySixStudio website built with React and GSAP.
This project showcases creative UI/UX, smooth animations, and scalable frontend architecture.

🚀 Features
Responsive Design: Works seamlessly across devices (desktop, tablet, mobile).

GSAP Animations: Timeline-based effects for navbar, hero section, and interactive elements.

Reusable Components: Modular structure for scalability and maintainability.

Creative Branding: Gradient backgrounds, glassmorphism, and cinematic UI effects.

Developer Friendly: Clean codebase with comments and clear folder structure.

🛠️ Tech Stack
Frontend: React.js

Styling: Tailwind CSS

Animations: GSAP (GreenSock Animation Platform)

Version Control: Git & GitHub
